01 Orbital Carousel

A face-forward 3D card carousel where cards travel a circular path using translateX + translateZ (not rotateY), so every card always faces the viewer. Three cards visible at once — front card full-scale, side cards at 72% — with drag, click-to-center, and 5-second auto-advance. The frosted glass background card rises from the bottom of the hero photo.

/* Scene — sets 3D perspective */ .carousel-scene { perspective: 900px; width: 240px; height: 370px; position: relative; transform-style: preserve-3d; } .carousel-cylinder { width: 240px; height: 370px; position: relative; transform-style: preserve-3d; } .cc-card { position: absolute; inset: 0; background: var(--white); border: 1px solid var(--border); border-radius: 2.2rem; opacity: 0; transition: transform .78s cubic-bezier(.25,.46,.45,.94), opacity .78s ease; } /* Frosted host card (hero glass lift) */ .hero-card { background: rgba(255,255,255,.10); border: 1px solid rgba(255,255,255,.18); backdrop-filter: blur(10px); -webkit-backdrop-filter: blur(10px); border-radius: clamp(36px,4.7vw,80px) clamp(36px,4.7vw,80px) 0 0; }
const TOTAL = 6; const STEP = 360 / TOTAL; // 60° const R = 300; // orbit radius px let totalAngle = 0; function positionCards(instant) { cards.forEach((card, i) => { const theta = ((i * STEP - totalAngle) % 360 + 360) % 360; const rad = theta * Math.PI / 180; const x = R * Math.sin(rad); const z = R * (Math.cos(rad) - 1); const front = theta === 0; const side = theta === 60 || theta === 300; card.style.transform = `translateX(${x}px) translateZ(${z}px) scale(${front ? 1 : side ? .72 : 0})`; card.style.opacity = front ? '1' : side ? '.52' : '0'; card.style.pointerEvents = (front || side) ? 'auto' : 'none'; card.style.zIndex = front ? 10 : side ? 5 : 1; }); } function rotate(delta) { totalAngle += delta; positionCards(); } // Auto-advance every 5s let timer = setInterval(() => rotate(+STEP), 5000); // Click side card to center it cards.forEach((card, i) => { card.addEventListener('click', () => { const theta = ((i * STEP - totalAngle) % 360 + 360) % 360; if (theta === 60) rotate(+STEP); if (theta === 300) rotate(-STEP); }); }); // Drag to rotate let dragX = null; scene.addEventListener('mousedown', e => dragX = e.clientX); window.addEventListener('mouseup', e => { if (dragX === null) return; const dx = e.clientX - dragX; if (Math.abs(dx) > 50) rotate(dx < 0 ? +STEP : -STEP); dragX = null; });

02 Glass Card

A frosted glass morphism card system built on backdrop-filter: blur(). Three tint variants — Light (10% white), Medium (20% white), Dark (55% navy) — for use over photo backgrounds. The current hero uses Light at 10px blur.

Three tint variants over photo
Light · 10% white · blur 10px
Glass Card Light
Subtle frost — photo reads through clearly. Used as the hero host card.
Medium · 20% white · blur 20px
Glass Card Medium
Higher opacity — dark text legible. Stronger material feel.
Dark · 55% navy · blur 18px
Glass Card Dark
Navy tint — strong contrast, photo still visible underneath.

03 Scroll Showcase

A scroll-jacked feature reveal: a 700vh tall scene with a sticky inner section. As the user scrolls, the tool list highlights, the browser frame crossfades to show the active tool's UI, and the headline text above crossfades to describe it. Uses grid-area: 1/1 overlay technique for zero-layout-shift crossfades.

/* Tall outer scene — enables scrolljack */ .sc-scene { height: 700vh; } /* Sticky inner — stays in view while scene scrolls */ .sc-section { position: sticky; top: 0; height: 100vh; display: flex; flex-direction: column; align-items: center; justify-content: center; padding: 60px clamp(30px,7vw,110px); background-color: #fbfcff; background-image: linear-gradient(to bottom, #fbfcff 0%, rgba(251,252,255,0) 38%), radial-gradient(circle, #c4d0e8 1px, transparent 1px); background-size: 100% 100%, 22px 22px; } /* Crossfade overlay technique — zero layout shift */ .sc-textstack { display: grid; } .sc-slide { grid-area: 1/1; /* all slides occupy same cell */ opacity: 0; transition: opacity .55s ease; pointer-events: none; } .sc-slide.active { opacity: 1; pointer-events: auto; } /* Tool list item */ .tl-item.active { background: rgba(90,233,215,.12); color: var(--navy); }
const scene = document.querySelector('.sc-scene'); const slides = document.querySelectorAll('.sc-slide'); const items = document.querySelectorAll('.tl-item'); const frames = document.querySelectorAll('.hf-card'); const TOTAL = slides.length; // 6 let current = 0; function showTool(idx, instant) { slides[current].classList.remove('active'); items[current].classList.remove('active'); frames[current].classList.remove('active', 'instant'); current = idx; slides[idx].classList.add('active'); items[idx].classList.add('active'); if (instant) frames[idx].classList.add('instant'); frames[idx].classList.add('active'); } // Sidebar click items.forEach(el => el.addEventListener('click', () => showTool(+el.dataset.sc, false)) ); // Scroll drive window.addEventListener('scroll', () => { const rect = scene.getBoundingClientRect(); const scrollable = scene.offsetHeight - window.innerHeight; const progress = Math.max(0, Math.min(1, -rect.top / scrollable)); const idx = Math.min(TOTAL - 1, Math.floor(progress * TOTAL)); showTool(idx, true); }, { passive: true });
