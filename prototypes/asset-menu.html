Rule: Mont 400 everywhere. No exceptions.
Every element in this project — headlines, body, captions, buttons, labels, nav — uses Mont 400 Regular. Impact comes from size, letter-spacing, and color, not weight. Bold type reads as loud; 400 reads as confident.
